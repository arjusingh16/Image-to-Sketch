ğŸ§© How It Works

1. The image is read using OpenCV.
2. Converted to grayscale.
3. Inverted to highlight lighter areas.
4. Blurred using a Gaussian filter for smooth shading.
5. The sketch is created using a division operation (NumPy) between the grayscale and inverted blurred image.
